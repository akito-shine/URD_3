# EAPlace

## PPA Results on the Ariane133 Benchmark
To evaluate the impact of the layout results generated by EGPlace on PPA metrics, we conducted experiments on the Ariane133 dataset, which is reported by ChipBench. The experimental flow is as follows: first, we selected the 256 largest modules from Ariane133 and performed placement using EGPlace. Then, we carried out mixed placement using DREAMPlace, followed by PPA evaluation using the code provided in ChipBench with mode "macro".
